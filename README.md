# The Lyrical Scholar — MBA Managerial Accounting Tutor

AI-powered personal tutor for MBA students at Texas A&M University-Central Texas.  
Built by Professor Anthony, Department Chair of Accounting & Finance.

## Features

- **8 Core Topics**: CVP Analysis, Master Budgeting, Variance Analysis, Product Costing, Decision Making, Performance Evaluation, Transfer Pricing, Forensic Accounting
- **Student Login & Progress Tracking**: Sessions, questions, quiz scores, mastery ratings
- **Instructor Dashboard**: View all student progress at a glance
- **CSV Export**: Export individual or all student data for Canvas gradebook
- **The Lyrical Scholar Methodology**: Named Principles, student-first explanations, real-world examples

## Quick Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Sign in with GitHub
3. Import this repository → Click Deploy
4. Share the URL with students via Canvas

## Local Development

```bash
npm install
npm run dev
```
